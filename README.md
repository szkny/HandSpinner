# HandSpinner
Simulation of Hand Spinner (Fidget Spinner) using GLUT and OpenGL

Extract the zip file and change directory to 'HandSpinner-master', then make 'HandSpinner'
```
$ cd ~/Downloads/HandSpinner-master
$ make
$ ./bin/HandSpinner
```
